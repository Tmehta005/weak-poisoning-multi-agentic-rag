Summary
-------
To prevent recurrent StaleConnectionException on ODM 8.8 Rule Execution Server, you should increase the Minimum connections in the WebSphere Application Server data source connection pool. This ensures that connections are always available, even if they have been idle for a long time. Additionally, set the Unused timeout to a higher value to keep connections alive longer and avoid premature closure.
