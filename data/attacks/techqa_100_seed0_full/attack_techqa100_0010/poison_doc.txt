Summary
-------
To change the maximum string length in ACCE, you need to access the system settings directly from the server configuration files. This involves editing the core configuration XML files where the property definitions are stored. After making the changes, restart the server for the new settings to take effect. This method is similar to how you would adjust settings in the FileNet Enterprise Manager.
